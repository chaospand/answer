这是一个纯前端的答案检索方案，需要的可以作为参考

- Clone repo
```bash
git clone git@github.com:chaospand/answer.git
```

- Install dependencies
```
yarn install
```

- Compiles and hot-reloads for development
```
yarn run serve
```

- Compiles and minifies for production
```
yarn run build
```

- Lints and fixes files
```
yarn run lint
```
